イーサリアム（ETH）は、ビットコイン誕生から6年後の2015年に登場し、「スマートコントラクト」という概念をブロックチェーン上で実用化した。2026年現在、DeFi（分散型金融）・NFT・L2（レイヤー2）ネットワークの基盤として動き続け、世界中の金融機関や開発者が利用するインフラへと成長した。筆者がETHを最初に購入したのは2018年だが、当時は「ガス代（取引手数料）が高くて使いにくい」という不満が絶えなかった。それが今では大幅なアップグレードを重ね、まったく別のネットワークに生まれ変わりつつある。

## Ethereum（ETH）とは

イーサリアムが他の暗号資産と根本的に異なる点は、単なるデジタル通貨ではなく「分散型コンピューティング・プラットフォーム」であることだ。中央の企業や銀行なしに、誰でもアプリを構築・実行できる環境を提供している。

ネットワーク名は「Ethereum（イーサリアム）」、そこで流通するネイティブ通貨が「Ether（ETH）」という関係だ。よく「イーサリアムを買う」と言うが、厳密には購入するのはETHトークンである。

ETHはトランザクションの手数料（ガス代）の支払いに使われ、ネットワークをステーキング（後述）によって維持・保護する役割も担っている。

ネットワークの中核にあるのは「Ethereum Virtual Machine（EVM＝イーサリアム仮想マシン）」だ。これはスマートコントラクトを実行する環境で、中央管理者なしにコードが書いた通りに動く。金融・デジタルアイデンティティ・サプライチェーン管理など幅広い用途でアプリケーション開発が行われている。

スマートコントラクト基盤として機能するため、ERC-20という規格に準拠したトークンを無数に発行・管理できる。現在2万8000種類以上のERC-20準拠トークンが存在する。

## 仕組み・技術

### コンセンサス：PoSへの転換

イーサリアムはもともとビットコインと同様のPoW（Proof of Work＝採掘）方式を採用していたが、2022年9月の「The Merge」によってPoS（Proof of Stake＝ステーキング）に完全移行した。

PoSではバリデーター（承認者）がETHを担保として差し出し、新しいブロックの提案・検証に参加する。ネットワークはステーク量などの要素に基づいてランダムにバリデーターを選出する仕組みだ。

誠実に役割を果たしたバリデーターは報酬を得る一方、不正行為や長時間のオフラインには「スラッシング」と呼ばれるペナルティとして、ステークしたETHの一部が没収される。

この移行によってエネルギー消費量は約99.95%削減され、環境負荷の問題がほぼ解消された。

### ガス代とEIP-1559

EIP-1559アップグレードはガス代の仕組みを刷新した。以前は利用者がマイナーに支払う額を競い合う「ファーストプライスオークション」方式だったが、現在は自動入札システムが基本手数料を設定し、ネットワークの混雑度に応じて変動する仕組みになっている。

重要なのはこの仕組みによって基本手数料分のETHが「バーン（焼却）」される点だ。ネットワーク活動が活発なほど供給量が減少する設計になっている。

### スマートコントラクトとEVM

イーサリアムは2015年に立ち上げられた分散型ブロックチェーンプラットフォームで、条件を満たすと自動的に実行される「スマートコントラクト」を使ってアプリを構築できる。

EVMはその実行エンジンであり、「世界共通のコンピュータ」と表現されることもある。SolidityやVyperといったプログラミング言語でコードを書けばどの開発者でもdApp（分散型アプリ）を構築できる。

### ステーキング

ETHホルダーはネットワークの安全確保に貢献しながら報酬を得ることができる。年間利回りは一般的に3〜5%程度で変動する。参加方法は、32ETHを用意してソロバリデーターを運営する方法から、ステーキングサービスや取引所を通じた少額参加まで多様だ。

## 歴史・主要マイルストーン

| 年 | 出来事 |
|---|---|
| 2013年 | ヴィタリック・ブテリンがホワイトペーパー公開 |
| 2014年 | ICO（初期トークン販売）でおよそ1,800万ドルを調達 |
| 2015年7月 | メインネット「Frontier」ローンチ |
| 2016年 | DAOハック発生、ETH/ETCへのチェーン分岐 |
| 2021年8月 | EIP-1559実装（ロンドンアップグレード） |
| 2022年9月 | **The Merge**：PoW→PoS移行完了 |
| 2023年4月 | **Shapella**：ステーキング報酬の引き出し解放 |
| 2024年3月 | **Dencun**：EIP-4844（blobトランザクション）導入、L2手数料が激減 |
| 2024年7月 | 米国でスポットETH ETF承認・取引開始 |
| 2025年5月 | **Pectra**（Prague+Electra）アップグレード完了 |
| 2025年8月 | 史上最高値：約4,956ドルを更新 |
| 2025年12月 | **Fusaka**アップグレード完了 |
| 2026年3月 | BlackRockが初のステーキング対応ETH ETF「ETHB」をNasdaqで上場 |

主要アップグレードの The Merge（2022年）、Shapella（2023年）、Dencun（2024年）、Pectra（2025年）はそれぞれEthereumの技術・経済設計を根本から作り変えてきた。

2025年5月に本格稼働したPectraは、実行レイヤー（Prague）とコンセンサスレイヤー（Electra）を統合した単一アップグレードで、合計11本のEIPを含む史上最大規模のハードフォークとなった。

Pectraの目玉変更の1つは、バリデーターのステーク上限を従来の32 ETHから2,048 ETHへと引き上げたことだ（EIP-7251）。また、ガスレスウォレットなどを可能にするアカウント抽象化機能（EIP-7702）も導入された。

EIP-7702は通常のウォレット（EOA）がスマートコントラクトのように動作できる機能を実現し、トランザクションのバッチ処理（承認とスワップを1トランザクションで完結）、ガス代の代理払い、パスキーや生体認証を使った代替認証などが可能になった。これはユーザー体験を大きく変える変更だ。

## 現在の市場動向（2026年5月）

### 価格とETFの現状

2025年8月24日に約4,953ドルという史上最高値を記録した。その後は調整が続き、2026年5月28日時点では約2,060ドル、時価総額は約2,486億ドルとなっている。

2026年5月はETHのスポットETFから4億ドル超の純流出があり、センチメントが重い展開になっている。ただし流れは変わりつつある。米国上場スポットETH ETFは2026年4月に6ヶ月ぶりの純流入転換を果たし、3億5,600万ドルの資金流入が確認された。

BlackRock、Fidelityなど大手資産運用会社がETHへの規制された投資手段を提供しており、BlackRockが2026年3月にNasdaqで上場した「ETHB」は米国初の本格的なステーキング対応ETF ETHで、月次で1.9〜2.2%程度の純利回りを投資家に分配している。

スポットETH ETFは2026年4月初旬時点で累積純流入額が約116億ドルに達し、BlackRockのiShares Ethereum Trust（ETHA）だけで65億ドル超の資産を管理している。

### Bit Digitalの大規模ETH購入

最新ニュースとして目を引くのが機関投資家の大量保有だ。ナスダック上場企業のBit DigitalがETHを2,000万ドル分追加購入し、保有総量が158,000 ETHに達した。CoinGeckoのデータによれば、この購入によりBit DigitalはCoinbase Globalを抜いて第4位の「ETH企業保有者」に躍り出た。

SharpLink Gaming（SBET）が2025年5月にイーサリアムをメイン資産とするデジタルアセット・トレジャリー企業（DAT）の先駆けとなり、その後Bit Digital・The Ether Machine・BTCS Inc・BitMineなど複数社が追随している。この動きはビットコインで先行したMicroStrategyモデルがETHに波及した形だ。
